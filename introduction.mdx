## Overview

Ben Smith provides a centralized platform where you create, organize, and manage all your project documentation. You build structured knowledge bases with Markdown support, version history, and search capabilities. Start documenting your projects immediately without complex setups.

<Callout kind="info">
  Ben Smith handles everything from API references to user guides in one space.
</Callout>

## Quick Navigation Guide

Get up and running with these essential steps.

<Steps>
  <Step title="Create Your Space" icon="plus">
    Sign up at `https://bensmith.com` and create a new documentation space for your project.
  </Step>
  <Step title="Add Your First Page" icon="file-plus">
    Click "New Page" and start writing in the Markdown editor.
  </Step>
  <Step title="Organize Content" icon="layers">
    Use the sidebar to nest pages and add navigation menus.
  </Step>
  <Step title="Publish and Share" icon="globe">
    Deploy your site and share the live URL with your team.
  </Step>
</Steps>

## Use Cases

Ben Smith adapts to various documentation needs.

<Tabs>
  <Tab title="Software Projects" icon="code">
    Document APIs, SDKs, and changelogs. Embed code examples like this:

    <CodeGroup tabs="JavaScript,Python">
    ```javascript
    const client = new BenSmithClient({ apiKey: `YOUR_API_KEY` });
    const docs = await client.getPages({ spaceId: `your-space-id` });
    console.log(docs);
    ```
    ```python
    from bensmith import Client
    client = Client(api_key="YOUR_API_KEY")
    docs = client.get_pages(space_id="your-space-id")
    print(docs)
    ```
    </CodeGroup>
  </Tab>
